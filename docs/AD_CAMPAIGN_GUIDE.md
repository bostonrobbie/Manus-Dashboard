## Performance Monitoring

### Key Metrics to Track

| Metric                   | Target | Action if Below                |
| ------------------------ | ------ | ------------------------------ |
| Click-Through Rate (CTR) | >2%    | Improve ad copy                |
| Cost Per Click (CPC)     | <$5    | Adjust bidding                 |
| Conversion Rate          | >3%    | Improve landing page           |
| Cost Per Conversion      | <$10   | Pause underperforming keywords |

### Weekly Review Process

1. Check Google Ads dashboard for spend and conversions
2. Review Umami analytics for scroll depth and time on page
3. Identify top-performing keywords
4. Pause keywords with high CPC and low conversions
5. A/B test new ad copy variations
