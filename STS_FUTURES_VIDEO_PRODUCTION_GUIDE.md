## Pre-Production Checklist

### Dashboard Setup

- [ ] Log into STS Futures dashboard as demo user
- [ ] Select "ALL" time range on all charts (shows 15-year data)
- [ ] Verify dark mode is enabled
- [ ] Populate My Dashboard with 8 subscribed strategies
- [ ] Pre-select ES Trend, NQ Trend, BTC Trend on Compare page

### Technical Setup

- [ ] Set display to 3840×2160 (4K) resolution
- [ ] Launch browser in full-screen mode (F11)
- [ ] Hide browser chrome, bookmarks bar, and extensions
- [ ] Set screen recording to 60fps, ProRes or high-bitrate MP4

### Audio Preparation

- [ ] Select and license music track (110 BPM recommended)
- [ ] Mark beat timestamps in audio editor
- [ ] Prepare sound effect library (bass hits, whooshes, UI sounds)
