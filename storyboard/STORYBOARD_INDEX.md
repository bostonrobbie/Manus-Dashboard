## Production Notes

### Color Palette

- **Background:** #0a1628 (dark navy)
- **Primary Accent:** #00d4ff (cyan)
- **Secondary Accent:** #00ff88 (green for positive values)
- **Text:** #ffffff (white)

### Typography

- **Headlines:** Bold sans-serif (Inter or similar)
- **Metrics:** Tabular numerals for alignment
- **Body:** Regular weight for descriptions

### Animation Guidelines

- All transitions should be smooth (ease-out-cubic)
- Metric counters animate from 0 to final value
- Glow effects pulse subtly on key elements
- Camera movements should feel cinematic, not robotic
